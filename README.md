## 功能与用途
用于智能化配置RoboMaster机甲大师裁判系统赛事引擎和图传IP设置，集成最新版MySQL8.0.28，且优化到整个文件只有16MB，简化繁琐的搭建和使用过程。
- 配置MySQL服务器
- 配置图传IP
- 配置RoboMaster裁判系统服务器运行环境
- 其他功能见下述 [更新说明](#更新说明)

## 支持的RM赛事引擎版本
- RMUA 所有版本
- RMUC 所有版本
- RMUL 所有版本
